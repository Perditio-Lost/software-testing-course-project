# 软件测试课程项目

[ [English](./README.md) | 中文 ]

## 项目简介

本仓库 fork 自项目：[OnlineExaminationSystem](https://github.com/liulang123xx/OnlineExaminationSystem)。

原项目是一个基于 JavaEE 的在线考试系统，主要提供用户管理、题库管理、试卷管理、在线考试、成绩管理、监考与防作弊等常见在线考试功能。

本仓库作为 **软件测试与质量保证课程** 的测试对象使用，主要用于记录测试设计、测试执行、测试结果与软件质量分析。

## 原系统简要说明

原系统主要包含以下模块：

- 用户认证与角色管理
- 题库管理
- 试卷管理
- 在线考试
- 成绩管理与成绩查询
- 在线监考与防作弊
- 统计分析与数据导出

本仓库重点不在于新增业务功能，而在于对原系统的部分核心模块进行软件测试与质量保证分析。

## 测试范围

本仓库中的测试工作主要包括：

- 成绩管理与成绩查询模块测试
- 监考系统与防作弊测试
- 成绩、批阅、监考、LiveKit 与防作弊相关服务的单元测试
- 成绩与监考相关接口测试
- 安全测试
- 最终测试结论与质量评价
